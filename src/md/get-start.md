# 开始使用

[安装](#/doc/install)完成之后。

在需要使用到的地方写入:
```js
import {Button, Tabs, Tab, Switch, Dialog, openDialog} from "oy-ui";

```
即可开始使用。


#### Like This:

```vue

<template>
  <Button>按钮一枚</Button>
</template>
<script>
  import { Button } from "oy-ui";
  export default {
    components: {
      Button
    }
  } 
</script>

```
