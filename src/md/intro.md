# 介绍

Oy-ui 是一款基于Vue3 和 TypeScript开发的 UI 组件库。

- 一致性
    - 与人们直觉相匹配的使用方法。
    - 所有元素的结构和风格保持一致。
    
- 反馈性
    - 页面的样式和交互可以有效的反馈给用户，让用户清晰的感知自己的动作。
    - 操作后页面的变化也清晰明了。
    
- 简洁
    - 配置简洁明了。
    - 语义化的配置项让用户能快速上手。
    
    


*目前已完成`Dialog`、`Button`、`Switch`、`Tabs`这四个组件，之后将继续补充。若有任何建议或意见欢迎[反馈](https://github.com/NightingaleM/oy-ui/issues)*
